# Data_cleaning_with_PySpark
Apache Spark DataFrame assignment demonstrating data loading, cleaning, filtering, transformations, aggregations, and basic ETL pipeline using PySpark.

# Apache Spark DataFrame Assignment

## Objective

Learn the basics of Apache Spark and perform data cleaning, transformation, filtering, and aggregation using Spark DataFrames.

## Tools Used

- Apache Spark
- PySpark
- Jupyter Notebook
- Python 3

## Dataset

A sample employee dataset containing employee information such as age, department, salary, and region.

## Tasks Performed

- Created a Spark Session
- Loaded CSV data into a DataFrame
- Viewed schema and sample records
- Removed duplicate rows
- Handled missing values
- Filtered data using multiple conditions
- Renamed columns
- Converted data types
- Performed aggregations
- Used groupBy() for departmental analysis
- Saved the final aggregated results

## Output

The final department-wise summary is stored in:

output/results.csv

## Key Concepts Learned

- Spark DataFrames
- Transformations and Actions
- Data Cleaning
- Filtering
- Aggregations
- Wide Transformations
- Shuffle Operations
